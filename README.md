# REIZE

A fast, multithreaded network discovery tool built in Python for pentesting and host discovery (Cyberstalker style).

## Features
- **High Performance**: Uses Python's `threading` and `queue` libraries for concurrent port scanning.
- **Clean CLI Interface**: Custom ASCII cyberpunk banner upon launch.
- **Easy Deployment**: Can be compiled into a `.deb` package for native Debian/Kali installation.

## Installation from Source
```bash
git clone https://github.com
cd Reize
python3 net_scanner.py
```
