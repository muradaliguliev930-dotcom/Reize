Установка REIZE на Linux (Kali Linux)

Следующий скрипт клонирует официальный репозиторий проекта, подтянет актуальные конфигурации и настроит бинарный пакет для вашей системы Linux:

git clone https://github.com && cd Reize && sudo dpkg -i reize_v5.5.0_amd64.deb

После успешной установки вы можете запустить интерактивную хакерскую консоль, введя команду reize из любого окна терминала.

Обновление REIZE до v5.5.0

Чтобы обновить утилиту до самой свежей закрытой сборки без потери локальных конфигурационных файлов и шаблонов, выполните форсированный пулл пакета:

cd ~/Reize && git pull origin main --force && sudo dpkg -i reize_v5.5.0_amd64.deb

Движок автоматически проверит целостность базы данных сигнатур и перезапустит фоновые процессы сканирования.

Архитектура и модули REIZE

Платформа работает по модульному принципу. При запуске основного ядра инициализируются три независимых подсистемы разведки:

  • Модуль DIR — 50-поточный скоростной фаззер скрытой веб-структуры.
  • Модуль VULN — Движок сигнатурного анализа уязвимостей по JSON-шаблонам.
  • Модуль INTEL — OSINT-разведка узлов, WHOIS и гео-трассировка IP.

Документация: Модуль DIR (DirBuster)

Предназначен для быстрого поиска скрытых административных панелей, логов и бэкапов на целевом сервере. Работает на базе низкоуровневых SSL-сокетов.

Пример использования команды:

reize dir ://vulnweb.com

Флаг автоматической очереди Queue распределяет запросы параллельно, исключая блокировку сетевого интерфейса Kali Linux.

Документация: Модуль VULN (Сканер Брешей)

Основной боевой движок платформы. Он считывает кастомные JSON-шаблоны из системной папки и проверяет заголовки, куки и ответы серверов на наличие критических уязвимостей.

Запуск сканирования уязвимостей:

reize vuln target-site.com

Вы можете расширять базу данных сканера, просто добавляя новые JSON-файлы сигнатур в каталог программы.
