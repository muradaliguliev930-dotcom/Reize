Установка REIZE на Linux (Kali Linux)

Следующий скрипт клонирует официальный репозиторий проекта, подтянет актуальные конфигурации и настроит бинарный пакет для вашей системы Linux:

git clone https://github.com && cd Reize && sudo dpkg -i reize_v5.5.0_amd64.deb

После успешной установки вы можете запустить интерактивную хакерскую консоль, введя команду reize из любого окна терминала. При первом запуске утилита автоматически развернет многопоточные движки и подготовит JSON-шаблоны уязвимостей.
